# EcoEasy 🌱

Projeto desenvolvido para a disciplina **Fábrica de Software**.

O **EcoEasy** é um sistema web simples que tem como objetivo auxiliar usuários no gerenciamento e descarte correto de resíduos. A aplicação permite consultar locais de descarte, solicitar coleta de materiais e registrar denúncias de descarte irregular.

---

## Funcionalidades

O sistema possui as seguintes funcionalidades:

- 🔐 Tela de Login
- 📍 Consulta de locais para descarte
- 🚛 Solicitação de coleta de resíduos e materiais
- ⚠️ Registro de denúncia de descarte irregular
- 📄 Geração de protocolos para solicitações e denúncias
